설치 방법

1

파일 다운로드 및 압축 해제

# 다운로드한 파일 압축 해제
tar -xzf auto-analyzer-v3-with-web-ui.tar.gz
# 디렉토리 이동
cd home/user/auto-analyzer-v3
2

Node.js 의존성 설치

npm install

Node.js 18 이상 필요 (node --version으로 확인)

3

Chrome 브라우저 설치 (필수)

# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y google-chrome-stable
# 또는 제공된 스크립트 사용
chmod +x install-chrome.sh
./install-chrome.sh
4

웹 인터페이스 실행

npm run web

브라우저에서 http://localhost:3000 열기

사용 방법

🌐 방법 1: 웹 인터페이스 (추천)

  1. npm run web 실행
  2. 브라우저에서 http://localhost:3000 열기
  3. 분석할 웹사이트 URL 입력 (예: https://www.naver.com)
  4. "분석 시작하기" 버튼 클릭
  5. 생성된 명령어를 새 터미널에서 실행

💻 방법 2: CLI 직접 실행

# JSON 리포트만
npm start -- analyze --url https://www.naver.com
# PDF 리포트 포함
npm start -- analyze --url https://www.naver.com --pdf

자주 묻는 질문

Q1. 왜 클라우드에서 실행 안 되나요?

이 도구는 실제 Chrome 브라우저를 실행하고, 웹사이트의 모든 버튼과 링크를 클릭하며 테스트합니다. Cloudflare Workers/Pages는 브라우저 실행이 불가능하므로, 본인의 컴퓨터에 설치하여 사용해야 합니다.

Q2. 분석 시간은 얼마나 걸리나요?

사이트 복잡도에 따라 다릅니다:

  • 간단한 사이트 (example.com): 약 35초
  • 중간 복잡도 (google.com): 약 45초
  • 복잡한 사이트 (naver.com): 약 90초

Q3. PDF 리포트에는 무엇이 포함되나요?

Nielsen 10대 원칙 레이더 차트, 43개 항목 상세 점수표, 상위/하위 5개 항목 분석, Lighthouse 성능 지표가 포함됩니다.
