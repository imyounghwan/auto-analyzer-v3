## 1. 설치 방법

### ✅ 준비물 확인
- **Node.js** v18 이상
- **Google Chrome** 브라우저
- **2GB 이상** RAM
- **500MB 이상** 디스크 공간

### 📥 1단계: 파일 다운로드
```
https://www.genspark.ai/api/files/s/3koTwgjB
```
브라우저에서 위 링크를 클릭하여 `auto-analyzer-v3-complete.tar.gz` 다운로드

### 📂 2단계: 압축 해제
```bash
# 다운로드한 파일이 있는 폴더로 이동
cd ~/Downloads

# 압축 해제
tar -xzf auto-analyzer-v3-complete.tar.gz

# 해제된 폴더로 이동
cd home/user/auto-analyzer-v3
```

### 📦 3단계: 의존성 설치
```bash
# Node.js 패키지 설치 (2-3분 소요)
npm install
```

**출력 예시:**
```
added 299 packages, and audited 353 packages in 2m
68 packages are looking for funding
found 0 vulnerabilities
```

### 🌐 4단계: Chrome 설치

#### **Ubuntu/Debian**
```bash
sudo apt-get update
sudo apt-get install -y google-chrome-stable
```

#### **또는 제공된 스크립트 사용**
```bash
chmod +x install-chrome.sh
./install-chrome.sh
```

#### **설치 확인**
```bash
google-chrome --version
```
**출력 예시:** `Google Chrome 145.0.7632.159`

## 3. 분석 실행

### 📝 1단계: URL 입력
웹 페이지 상단 입력란에 분석할 웹사이트 주소 입력

**예시:**
- `https://www.naver.com`
- `https://www.google.com`
- `https://www.yes24.com`
- `https://www.gmarket.co.kr`

### ⚙️ 2단계: 옵션 선택

#### **PDF 리포트 생성** (권장 ✅)
- 체크하면: JSON + **PDF 파일** 생성 (레이더 차트 포함)
- 체크 안 하면: JSON 파일만 생성

#### **캐시 사용 안 함**
- 체크하면: 항상 새로 분석 (느림)
- 체크 안 하면: 1시간 이내 재분석 시 캐시 사용 (빠름)

### 🎬 3단계: 분석 시작 버튼 클릭
"🚀 분석 시작하기" 버튼 클릭!

**✨ 이제 자동으로 분석이 시작됩니다!**
- 버튼 클릭 후 30-40초 대기
- 진행 상태 바가 표시됨
- 완료되면 자동으로 결과 페이지로 이동

**진행 화면:**
```

│  🔄 분석 진행 중...                          │
│  예상 시간: 약 30-40초                       │
```

### ⏳ 4단계: 자동 완료 대기
**더 이상 터미널 작업 필요 없음!**

## 5. 문제 해결

### ❌ 문제 1: `npm install` 실패
**증상:**
```
npm ERR! code EACCES
npm ERR! syscall access
```

**해결:**
```bash
# 권한 문제 - 폴더 소유권 변경
sudo chown -R $USER:$USER .

# 다시 설치
npm install
```

### ❌ 문제 2: Chrome이 없다고 나옴
**증상:**
```
Error: Could not find Chrome
```

**해결:**
```bash
# Chrome 설치 확인
google-chrome --version

# 없으면 설치
sudo apt-get install -y google-chrome-stable

# 또는
./install-chrome.sh
```

### ❌ 문제 3: 포트 3000이 이미 사용 중
**증상:**
```
Error: listen EADDRINUSE: address already in use :::3000
```

**해결:**
```bash
# 3000 포트 사용 프로세스 종료
fuser -k 3000/tcp

# 또는
lsof -ti:3000 | xargs kill -9

# 다시 실행
npm run web
```

### ❌ 문제 4: 분석이 멈춤
**증상:**
터미널에서 진행률이 더 이상 안 올라감

**해결:**
```bash
# Ctrl+C로 중단

# Chrome 프로세스 모두 종료
pkill -9 chrome

# 다시 실행
npm start -- analyze --url <URL> --pdf --no-cache
```

### ❌ 문제 5: 결과 페이지가 안 열림
**증상:**
```
http://localhost:3000/result.html?file=...
→ 404 Not Found
```

**해결:**
```bash
# 웹 서버가 실행 중인지 확인
curl http://localhost:3000

# 안 되면 웹 서버 재시작
npm run web

# 파일명이 정확한지 확인
cd output
ls -l *.json
```

```bash
# 웹 서버 시작
npm run web

npm start -- analyze --url <URL>

# 분석 실행 (PDF 포함)
npm start -- analyze --url <URL> --pdf

# 분석 실행 (캐시 안 씀)
npm start -- analyze --url <URL> --pdf --no-cache

# 결과 폴더로 이동

ls -lht output/

# 웹 서버 종료
```
